# NYC-Taxi and Limousine Commission

Para el tetramestre, se utilizara la base de datos de : [NYC-Taxi and Limousine Commission](https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page). 

Unicamente se utilizaran los datos correspondientes a los taxis amarillos del año 2024 (No se selecciono el año del 2025 por que no estaban completos).

# Pyspark 
Se utilizara pyspark de manera local mediante Visaul Studio Code (se tuvo que bajar la version de Python a la 3.11.9 para que funcione bien con Apache Spark).
